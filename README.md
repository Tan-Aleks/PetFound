# ПетПоиск Москва

Платформа для поиска пропавших домашних питомцев и размещения объявлений о найденных животных в Москве.

## Особенности

- 🔍 **Умный поиск** - поиск по фото с помощью ИИ, расширенные фильтры
- 🗺️ **Геолокация** - поиск по районам Москвы
- 📱 **Уведомления** - SMS, звонки и email при совпадениях
- 👥 **Волонтерская сеть** - активное сообщество помощников
- 🔒 **Безопасность** - защищенная система общения
- 🌐 **Интеграция** - мониторинг других сайтов объявлений

## Технологический стек

- **Frontend**: Next.js 15, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Supabase (PostgreSQL)
- **Authentication**: NextAuth.js
- **Deployment**: Vercel
- **Package Manager**: pnpm

## Быстрый старт

### Предварительные требования

- Node.js 18+ 
- pnpm (рекомендуется) или npm
- Аккаунт Supabase
- Git

### Установка

1. Клонируйте репозиторий:
\`\`\`bash
git clone <repository-url>
cd pet-search-moscow
\`\`\`

2. Установите зависимости:
\`\`\`bash
pnpm install
\`\`\`

3. Настройте переменные окружения:
\`\`\`bash
cp .env.local.example .env.local
\`\`\`

Заполните файл \`.env.local\` вашими данными:
\`\`\`env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your_nextauth_secret
\`\`\`

4. Настройте базу данных Supabase:
   - Создайте новый проект в [Supabase](https://supabase.com)
   - Выполните SQL из файла \`supabase/schema.sql\` в SQL Editor
   - Настройте RLS политики

5. Запустите проект:
\`\`\`bash
pnpm run dev
\`\`\`

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Основные функции

### Поиск питомцев
- Расширенный поиск по параметрам (тип, порода, район, дата)
- AI-поиск по фотографии
- Интеграция с внешними источниками

### Размещение объявлений
- Объявления о пропавших питомцах
- Объявления о найденных животных
- Загрузка фотографий
- Система вознаграждений

### Коммуникация
- Внутренний чат между пользователями
- Уведомления по SMS и email
- Система волонтеров

### Безопасность
- Аутентификация пользователей
- Модерация контента
- Защита персональных данных

## Разработка

### Команды

\`\`\`bash
# Разработка
pnpm run dev

# Сборка
pnpm run build

# Запуск продакшн версии
pnpm run start

# Линтинг и форматирование
pnpm run lint
pnpm run lint:fix
pnpm run format
\`\`\`

### Настройка Supabase

1. **Создание таблиц**: Выполните SQL из \`supabase/schema.sql\`
2. **RLS политики**: Настроены автоматически в схеме
3. **Storage**: Настройте bucket для фотографий питомцев
4. **Edge Functions**: Для AI-анализа фотографий (опционально)

### Интеграция с внешними API

Для мониторинга других сайтов объявлений:
- Настройте парсеры в \`lib/parsers/\`
- Добавьте cron jobs для регулярного обновления
- Настройте AI для сопоставления объявлений
