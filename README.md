# ПетПоиск Москва

Платформа для поиска пропавших домашних питомцев и размещения объявлений о найденных животных в Москве.

## Особенности

- 🔍 **Умный поиск** - поиск по фото с помощью ИИ, расширенные фильтры
- 🗺️ **Геолокация** - поиск по районам Москвы
- 👥 **Волонтерская сеть** - активное сообщество помощников
- 🔒 **Безопасность** - защищенная система общения

## Технологический стек

- **Frontend**: Next.js 15, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Supabase (PostgreSQL)
- **Package Manager**: **Bun** (основной)
- **Deployment**: Vercel

## Быстрый старт

### Предварительные требования

- **Bun** (установлен локально)
- Аккаунт Supabase
- Git

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Tan-Aleks/PetFound.git
cd PetFound
```

2. Установите зависимости:
```bash
bun install
```

3. Настройте переменные окружения:
```bash
cp .env.local.example .env.local
```

4. Запустите проект:
```bash
bun dev
```

## Структура проекта

```
PetFound/
├── app/                    # Next.js App Router (Корневые страницы)
├── components/            # React компоненты
│   └── ui/               # UI компоненты (shadcn/ui)
├── docs/                  # Документация (ТЗ и инструкции)
├── hooks/                # Custom React hooks
├── lib/                  # Утилиты и клиент Supabase
├── supabase/             # SQL схемы (schema.sql)
├── public/               # Статические файлы
└── bun.lock              # Lock-файл Bun
```
