# Docs Overview

## Architecture Source of Truth
Этот файл является каноническим описанием высокоуровневой архитектуры проекта `ПетПоиск Москва`.

## Product Scope
- Платформа для публикации объявлений о пропавших и найденных питомцах.
- География MVP: Москва и Московская область.
- Основные пользовательские сценарии MVP: регистрация, просмотр объявлений, поиск, создание объявления с фото, чат по объявлению.

## System Architecture
- `app/`: Next.js App Router, страницы и route handlers.
- `components/`: переиспользуемые UI-компоненты и каркасные элементы интерфейса.
- `hooks/`: клиентские хуки для загрузки объявлений и чатов.
- `lib/`: интеграции с Supabase, NextAuth и служебные утилиты.
- `supabase/schema.sql`: SQL-схема, RLS-политики и настройки bucket `pet-photos`.
- `memory_bank/`: канонический операционный контекст проекта, прогресс и модульная документация.

## Data Access Model
- Публичное чтение объявлений проходит через Next.js API-маршруты.
- Чувствительные операции записи и приватное чтение сообщений выполняются через server-side route handlers.
- Аутентификация построена на `NextAuth` с использованием `Supabase Auth`.
- Для серверных операций используется service-role клиент Supabase.
- Realtime-подписка чата остается на браузерном клиенте Supabase и получает обновления из PostgreSQL Realtime.

## Public Routes
- `/`: главная страница и последние объявления.
- `/search`: поиск и фильтрация объявлений (включая AI-поиск по фото).
- `/create`: создание объявления.
- `/pet/[id]`: детальная карточка питомца.
- `/chat`: список диалогов пользователя.
- `/chat/[petId]`: чат по объявлению.
- `/external`: внешние источники и партнерские объявления.
- `/login`, `/register`: вход и регистрация.

## AI-поиск по изображению
- Используется CLIP модель (`Xenova/clip-vit-base-patch16`) через `@xenova/transformers`.
- API endpoint: `POST /api/pets/search-by-image`
- Поддерживаемые форматы: JPEG, PNG, WebP.
- Порог схожести: 25%.

## Current Boundaries
- Платформа закрывает все основные сценарии: объявления, поиск (текстовый + AI), аутентификацию, чат, уведомления и внешние источники.
- Проект завершен на 100%.
