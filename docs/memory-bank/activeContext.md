# Active Context

## Current Status
MVP перешел из стадии инфраструктурной подготовки в рабочую стадию: аутентификация, реальные данные Supabase, создание объявлений с фото и базовый чат уже реализованы.

## Recent Changes
- **Данные**: Главная и поиск переведены с моков на реальные запросы к Supabase.
- **Аутентификация**: Подключены `NextAuth` и `Supabase Auth` для регистрации/входа.
- **Создание объявления**: Реализована запись в `pets` и загрузка изображений в bucket `pet-photos`.
- **Чат**: Добавлена страница `/chat/[petId]` с чтением и отправкой сообщений из таблицы `messages`.
- **UI/UX**: Внедрено переключение светлой/темной темы и обновлены токены дизайна.

## Next Steps

2. Проверить production-политики и доступ к bucket `pet-photos`.
