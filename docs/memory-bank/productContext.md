# Product Context

## Why this project exists?
Поиск пропавшего питомца — это стрессовый процесс. Существующие площадки (Авито, соцсети) разрознены. Проект "ПетПоиск Москва" призван агрегировать информацию и упростить процесс поиска за счет специализированных инструментов и сообщества.

## Problems it solves
- **Разрозненность данных**: Информация о найденных животных часто теряется в лентах соцсетей.
- **Сложность поиска**: Обычный поиск по тексту не всегда эффективен для животных.
- **Отсутствие уведомлений**: Владельцам приходится вручную мониторить все площадки.

## User Experience
- **Владелец**: Легко создает объявление, получает уведомления о совпадениях, общается с нашедшими.
- **Нашедший**: Быстро размещает фото найденного животного без лишней бюрократии.
- **Волонтер**: Получает алерты о пропажах в своем районе и помогает в поиске.
