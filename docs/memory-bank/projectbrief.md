# Project Brief: ПетПоиск Москва

## Core Objective
Создание веб-платформы для поиска пропавших домашних питомцев и размещения объявлений о найденных животных в Москве и Московской области.

## Goals
- Предоставление удобного инструмента для воссоединения владельцев с их питомцами.
- Использование AI для поиска по фото (в будущем).
- Интеграция с внешними источниками объявлений.
- Создание активного волонтерского сообщества.

## Scope
- Регистрация и аутентификация пользователей.
- Объявления типов "Пропал" и "Найден".
- Система поиска и фильтрации.
- Внутренний чат и уведомления (SMS/Email).
- Бесплатная социальная инициатива.
