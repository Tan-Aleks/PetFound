# Active Context

## Current Focus
- Канонический `memory_bank` перенесен в корень репозитория в соответствии с новыми правилами `AGENTS.md`.
- High-level архитектура зафиксирована в `docs/README.md`.
- Публичное чтение объявлений и приватное чтение сообщений переведены на server-side API, чтобы уменьшить прямой доступ браузера к чувствительным данным.

## Current Status
- MVP уже покрывает регистрацию, просмотр и поиск объявлений, создание объявлений с фото, детальную карточку питомца и чат по объявлению.
- Для чтения данных используется смешанная модель: route handlers для списков/деталей/диалогов и браузерный Supabase client для realtime-подписки чата.
