# Project Brief: ПетПоиск Москва

## Core Objective
Создание веб-платформы для поиска пропавших домашних питомцев и размещения объявлений о найденных животных в Москве и Московской области.

## Goals
- Сократить время поиска питомцев за счет специализированного каталога объявлений.
- Дать пользователям безопасный канал связи внутри платформы.
- Подготовить архитектуру к следующим этапам: уведомления, AI-поиск, внешние источники, волонтерские сценарии.

## Scope
- Регистрация и аутентификация пользователей.
- Размещение объявлений типов `lost` и `found`.
- Поиск и фильтрация объявлений.
- Загрузка фотографий питомцев.
- Внутренний чат по объявлению.
