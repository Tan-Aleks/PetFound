# Shared Components

## Header
- Основная навигация между публичными разделами.

## SearchForm
- Единая форма фильтрации для главной и поиска.
- Синхронизируется с query string и хуками объявлений.

## PetCard
- Краткое представление объявления в каталоге.

## ImageSearchUploader
- Загружает фото для AI-поиска и показывает состояние анализа изображения.

## ThemeToggle
- Переключает светлую и темную тему.

## Providers
- Подключает глобальные провайдеры приложения.

## NotificationDropdown
- Показывает уведомления пользователя, умеет помечать их прочитанными и для `match_found` дает отдельные действия: переход к внутреннему объявлению, переход к внешнему совпадению и ссылку на сайт-источник.

## AiMatchChatPanel
- Показывает системный AI-чат по внешнему совпадению, визуально встроенный в страницу обычного чата, но с отдельным контуром данных и жесткими ограничениями тем вопросов.
