# Profile Page (Личный кабинет)

## Описание
Страница личного кабинета пользователя с просмотром и редактированием профиля, а также управлением своими объявлениями.

## Роут
- `/profile` - страница личного кабинета (требует авторизации)

## Компоненты
- `ProfileClient` (`app/profile/ProfileClient.tsx`) - клиентский компонент со всей логикой

## API Endpoints
- `GET /api/profile` - получение данных профиля пользователя
- `PUT /api/profile` - обновление данных профиля (name, phone, district)
- `GET /api/profile/pets` - получение объявлений текущего пользователя
- `DELETE /api/profile/pets/[id]` - удаление объявления

## Потоки данных

### Загрузка данных
1. При монтировании загружаются профиль и объявления пользователя параллельно
2. Данные кешируются в локальном состоянии

### Редактирование профиля
1. Пользователь нажимает "Редактировать"
2. Появляется форма с текущими данными
3. После сохранения данные отправляются PUT /api/profile
4. При успехе обновляется состояние и вызывается updateSession()

### Удаление объявления
1. При наведении на карточку появляется кнопка "Удалить"
2. После подтверждения отправляется DELETE /api/profile/pets/[id]
3. При успехе объявление удаляется из списка

## Связь с Header
Header обновлен для отображения:
- Ссылки на профиль вместо кнопок "Войти/Регистрация" для авторизованных
- Диалоги показываются только авторизованным
