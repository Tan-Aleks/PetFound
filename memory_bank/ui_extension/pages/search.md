# Page: Search

## Route
- `/search`

## Purpose
- Предоставляет расширенный поиск объявлений.

## Main Components
- `Header`
- `SearchForm`
- `PetCard`
- `ImageSearchUploader`

## Data Flow
- Параметры поиска читаются из query string.
- Результаты загружаются через `useSearchPets()` -> `GET /api/pets`.
- AI-поиск по фото работает через `useImageSearch()` -> `POST /api/pets/search-by-image`.
- В режиме поиска по фото страница показывает две группы результатов: внутренние объявления платформы и внешние совпадения из партнерских источников.
- Во время AI-поиска сервер параллельно обновляет `cross_matches`, чтобы найденные внешние совпадения можно было использовать для автоматических уведомлений.
