# Volunteers Page (Волонтеры)

## Описание
Страница для регистрации и управления статусом волонтера. Волонтеры выбирают районы, в которых готовы помогать с поиском питомцев.

## Роут
- `/volunteers` - страница управления волонтерством (требует авторизации)

## API Endpoints
- `GET /api/volunteers` - получение данных волонтера
- `POST /api/volunteers` - регистрация/обновление волонтера с районами
- `DELETE /api/volunteers` - отмена регистрации волонтера

## Компоненты
- `Badge` (`components/ui/badge.tsx`) - компонент для отображения выбранных районов

## Потоки данных
1. При загрузке проверяется авторизация, редирект на /login если не авторизован
2. Загружаются данные волонтера (если есть)
3. Пользователь добавляет/удаляет районы из списка
4. Сохранение отправляет POST с массивом районов
5. Отмена регистрации удаляет запись волонтера

## Районы Москвы
Список районов определен в константе `MOSCOW_DISTRICTS`:
- Центральный, Северный, Северо-Восточный, Восточный, Юго-Восточный, Южный, Юго-Западный, Западный, Северо-Западный, Зеленоградский округ
